Display author information under post.

== Description ==

Displaying author info by Author Box is super easy and customizable. use different themes to match with your site template.



<strong>Plugin Features</strong>

* Different themes.
* Social links for autors.
* Responsive design.
* Unlimited user profile field.
* Control social icon on author box.
* Margin for social icons.
* Background color for author box area.
* Filter post-type to display author box.




== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and <strong>activate</strong> it.
3. After Activate you see Menu on left Sidebar "<strong>Author Box</strong>" 

if you are first time change settings and hit <strong>"Save Changes"</strong>

and visit your website or single post.
